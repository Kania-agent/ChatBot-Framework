# ChatBot Framework

Multi-LLM chatbot interface powered by MiMo V2.5.

## Features

- Modern chat UI with message bubbles
- Model selector dropdown (MiMo V2.5, MiMo-V2, Claude, GPT-4, Llama)
- Typing indicators and auto-scroll
- Message timestamps
- Responsive design

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6+)
- No external dependencies or frameworks

## Usage

Open `index.html` in a browser. Select a model from the dropdown and start chatting. The bot simulates AI responses with typing indicators.

## AI Engine

Built on **MiMo V2.5** — Nous Research's conversational AI model.
